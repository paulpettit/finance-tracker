How to export CSVs

🏦 Chase

Log into Chase.com → Select account → Click the download icon (↓) → Choose CSV → Select date range → Download

💳 American Express

Log into Amex → Statements & Activity → Download your statement data → Choose CSV → Select date range

📊 Fidelity

Log into Fidelity → Accounts → Activity & Orders → Download (top right) → CSV

🪙 Coinbase

Log into Coinbase → Click profile icon → Taxes → Generate Report → Transaction History CSV

🪶 Robinhood

Log into Robinhood → Account → Statements & History → Download account statements as CSV

🏛️ Marcus

Log into Marcus → Select account → Transaction History → Download Transactions → CSV

📈 Charles Schwab

Log into Schwab → Accounts → History → Export (top right) → CSV
