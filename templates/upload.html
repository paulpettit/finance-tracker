How to export from your bank

🏦 Chase

Chase.com → Select account → Download icon (↓) → CSV → Select date range

💳 American Express

Amex → Statements & Activity → Download → CSV → Date range

📊 Fidelity

Fidelity → Accounts → Activity & Orders → Download → CSV

🪙 Coinbase

Coinbase → Profile → Taxes → Generate Report → Transaction History CSV

🪶 Robinhood

Robinhood → Account → Statements & History → Download CSV

🏛️ Marcus

Marcus → Account → Transaction History → Download → CSV

📈 Charles Schwab

Schwab → Accounts → History → Export → CSV
